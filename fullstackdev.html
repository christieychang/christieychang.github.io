During the 67-272 course, I built a Ruby On Rails website for chess camps. The users that could use the website are administrators, chess instructors and parents/guardian who would sign their children up for the camps. While one of the main objectives of this course was to create a fully functioning web application, we also focused on user-centered design for all three user types.
